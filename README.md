#  readaratus
####  A dynamic book reading system for computers
## Features
- Unit conversion
- Figure referencing
- Teleportation
- Enhanced TOC
- Dual page search
- TOC synthesis for books that lack one
- Real page numbers
## Features being developed
- Table to chart
- Live code snippets 
- Keyboard-averse gesture commanding

## Compile
### Dependencies
 - Poppler's GLib backend
 - Gtk3 SDK
 ###
 ```bash
 make
 ./readaraus
 ```
