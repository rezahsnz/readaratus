# readaratus

A dynamic book reading system for computers.
## Features
- Unit conversion
- Figure referencing
- Teleportation
- Enhanced TOC
- Dual page search
## Features being developed
- Table to chart
- TOC synthesis for books that lack one
- Live code snippets 
- Keyboard-averse gesture commanding
